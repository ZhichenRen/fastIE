FastIE 中文文档
=====================

`FastIE <https://github.com/open-nlplab/fastIE>`_ 是一款信息抽取通用集成框架。您既可以快速利用 FastIE 中的海量模型对您的数据进行训练、检验、推理，又可以快速基于现有模型修改，创建自己的模型。

FastIE 具有如下的特性：

- 简化训练、检验、推理过程，SOTA 模型开箱即用；
